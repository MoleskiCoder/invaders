## Invaders

Space Invaders Emulator written in C++

Uses SDL2 to provide graphics and audio processing.

## Compiling

### Windows

Compile with Visual Studio 2015 (via the solution)

## Running

### Keyboard Controls

* 3: Insert Coin
* 1: 1P Start
* 2: 2P Start
* z: Left 1P
* x: Right 1P
* \: Fire 1P
