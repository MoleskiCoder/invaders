## Invaders

Space Invaders Emulator written in C++

Uses SDL2 to provide graphics and SDL_mixer for sound effects.

### Features

* 2 player controls
* Coloured gel screen
* Sound effects
* Event driven Intel 8080 emulator
* Basic CP/M emulation (enough to run CPU tests)
* Intel 8080 profiler
* Intel 8080 disassembler

### Keyboard Controls

* 3: Insert Coin
* 1: 1P Start
* 2: 2P Start
* z: Left 1P
* x: Right 1P
* \\: Fire 1P
* ,: Left 2P
* .: Right 2P
* /: Fire 2P
